# Google Web Search

## Available Tools

- **google-search___google_web_search(query, include_images)**: Search the web using Google Custom Search.
  - `query` (string, required): Search query
  - `include_images` (bool, default: true): Include image results

## Usage Guidelines
- Formulate clear, specific search queries for best results.
- Use quotes for exact phrase matching (e.g., `"machine learning" best practices 2025`).
- For comprehensive research, perform multiple targeted searches.
- Set include_images=false for text-only searches.
